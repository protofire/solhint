## Description
Explicitly mark all external contracts as trusted or untrusted.

## Options
This rule accepts a string option for rule severity. Must be one of "error", "warn", "off". Defaults to warn.

### Example Config
```json
{
  "rules": {
    "mark-callable-contracts": "warn"
  }
}
```
